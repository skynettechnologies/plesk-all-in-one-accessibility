## Installation

### Prerequisites
- Plesk version v18.0

### Steps

You can install the All in One Accessibility extension in two ways:

### Option 1: Plesk Marketplace
1. Log in to your Plesk control panel.
2. Navigate to **Extensions**.
3. Search for *All in One Accessibility*.
4. Click **Install** to add the extension to your Plesk environment.

### Option 2: Upload via Extension Manager
1. Log in to your Plesk control panel.
2. Navigate to **Extensions** > **My Extensions**.
3. Click on **Upload Extension**.
4. Choose the ZIP file of the All in One Accessibility extension from your local machine.
5. Click **OK** to upload and install the extension.

## Activation and Deactivation
1. Navigate to **Extensions** > **My Extensions**.
2. Locate *All in One Accessibility* in the list.
3. Use the **Enable** or **Disable** toggle to activate or deactivate the extension as needed.


## Configuration

1. Go to **Extensions** > **My Extensions**.
2. Locate *All in One Accessibility* and click **Open**.
3. In the settings page:
   - **Select the domain** where you want the widget settings to apply.
   - Adjust the widget settings as per your requirements.
4. Click **Save** to apply your changes.
5. Visit the selected domain to verify that the widget is visible on the frontend.

## CORS Policy Configuration

To avoid CORS policy issues, ensure the following URLs are allowed in your website. These URLs should be added to your CORS configuration or trusted domains list.

| **Domain**                        | **Description**                              | **Usage**                        |
|-----------------------------------|----------------------------------------------|----------------------------------|
| `https://*.skynettechnologies.com` | Skynet Technologies (Global Domain)         | API access and resources        |
| `https://*.skynettechnologies.us` | Skynet Technologies (US Domain)             | API access and resources        |
| `https://*.googleapis.com`        | Google APIs                                 | Services like Fonts, Translation |
